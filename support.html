Where is my data stored?

All app data is stored locally on your iPhone. MathFeedback does not upload student records or feedback reports to a developer server.

How do I move data to a new phone?

Open Settings in MathFeedback, export all data as a JSON backup, save it to Files or iCloud Drive, then import the backup on the new phone.

What should I do before updating or reinstalling?

Export a backup from Settings first. Keep the backup file in a safe place.

How do I report a problem?

Email the support address above and include:

  • iOS version
  • MathFeedback version
  • A short description of the problem
  • Steps to reproduce the issue, if possible
